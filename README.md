# acervenky.github.io

Personal portfolio and hiring profile for Venkatesh Surve.

## Structure

```
index.html        → acervenky.github.io          (personality-first, full scope)
hiring/
  index.html      → acervenky.github.io/hiring   (recruiter-optimised, conversion-focused)
llms.txt          → acervenky.github.io/llms.txt  (machine-readable profile for LLM crawlers)
```

## Stack

Vanilla HTML, CSS, and JavaScript. No frameworks, no build step, no dependencies.
